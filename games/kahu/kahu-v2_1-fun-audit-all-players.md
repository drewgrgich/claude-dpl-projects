# Kahu v2.1 — Fun Audit Results (All Player Counts)

*500 AI-vs-AI simulations per player count. Expert-level heuristic AI. March 2026.*

---

## v2.1 Rules Summary

Changes from v1.0 (rules as written):

- **Offering cooldown** — 1 full round between Offering completions, waived at lava ≤ 5
- **Cheaper Tikis** — Cost 1R+1B (was 1R+1B+1Y)
- **Surf VP threshold** — Surf worth 1 VP each only if you own 4+
- **Seat compensation** — P1, P2, P3 each start with 1 random Pua (P0 gets nothing)
- **VP tokens** — Unchanged at [4, 3, 2, 1]
- **Lava track** — Unchanged (2P: 16, 3P: 20, 4P: 22)

The tension curve at C reflects VP front-loading from early Pua-fueled completions.

## Recommendations

The strongest v2.1 configuration may be offering cooldown + cheaper Tikis + Surf threshold **without** the seat compensation Pua. In earlier testing, that combination achieved Overall A at 4P with tension B (49% ascending) and blowout C (12%) — both improvements over v1.0 — without the blowout regression seen here. The first-player advantage would remain, but the fun metrics would be cleaner.

If addressing seat order is a priority, the compensation mechanism needs to avoid the Pua/scoring axis. Options to explore in physical playtesting: extra card draw for later seats, first pick from a wider opening market, or a bid-for-seat-order mechanic.
