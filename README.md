# Design Play Labs — Game Design Projects

This repo contains the working files for the Hamsters & Monsters game system and other Design Play Labs projects.

## Structure

- `games/monster-mixer/` — Monster Mixer card game iteration files, simulations, and rules

## About Hamsters & Monsters

A 66-card game system that plays multiple games using the same deck. Designed in collaboration with the Designer Corps AI system.

More info: [designplaylabs.com](https://designplaylabs.com)
