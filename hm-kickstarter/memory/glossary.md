# Glossary

Hamsters & Monsters project shorthand, acronyms, and internal language.

## Acronyms
| Term | Meaning | Context |
|------|---------|---------|
| H&M | Hamsters & Monsters | The card game |
| DPL | Design Play Labs | Drew's game company |
| KS | Kickstarter | Crowdfunding platform |
| KS1 | First Kickstarter campaign | Target: March 9, 2027 (was Sept 1, 2026 — delayed due to hold) |
| KS2 | Second Kickstarter campaign | 36-month option window after KS1 fulfillment |
| MOQ | Minimum Order Quantity | 1,500 units for manufacturing |
| RFQ | Request for Quote | Manufacturer pricing requests |
| BGG | BoardGameGeek | Major board game community site |
| PnP | Print and Play | Free downloadable prototype for email capture |
| CTA | Call to Action | Marketing term |
| LP | Landing Page | Email capture page |
| UGC | User Generated Content | Fan art, playtest photos, etc. |
| BTS | Behind the Scenes | Content type for social media |

## Game Terms
| Term | Meaning |
|------|---------|
| Hammies | The hamsters in the game |
| Cohorts | Teams of 2 (1 Hammie + 1 Monster) |
| Factions | The 6 suit-based groups |
| Finders-Keepers | Red faction — Aggression / First Pick |
| Munchies | Orange faction — Gluttony / Resources |
| Tinkerers | Yellow faction — Speed / Cycling |
| Prognosticationers | Green faction — Vision / Scouting |
| Magicians | Blue faction — Trickery / Theft |
| Time Travelers | Purple faction — Mystery / Recursion |
| Get Stuffed | One of the games playable with the deck (possible PnP candidate) |
| Master Codex | Full game design document in Google Drive |

## Business Terms
| Term | Meaning |
|------|---------|
| Net Receipts | KS cash received minus platform fees, processing fees, and refunds — base for Danny's royalty |
| Profit-share kicker | Bonus payments to Danny at funding thresholds ($50K/$75K/$150K) |
| Smoke test | First Danny IG post linking to landing page — tests audience conversion |
| Blind playtest | Someone learns the game from rulebook alone with no guidance |
| Promo deck | Prototype deck with temp/placeholder art |
| Sell-off period | 12 months post-termination to sell remaining inventory |
